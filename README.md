# MonkeyQuest
This is a modified version of the MonkeyQuest addon. I combined MonkeyLibrary, MonkeyBuddy, and MonkeySpeed into a single addon. I also integrated it with pfQuest somewhat and cleaned up and rewrote a lot of the functions. I much prefer MonkeyQuest for keeping track of quests/objectives than any other quest addon I've ever used. Especially for sharing(ctrl-left click) and abandoning quests(ctrl-right click).  

PS: For pfQuest integration, left-click to show quest objectives. Right-click to hide quest objectives. Shift-left click to block opening the world map when showing quest objectives. Enables quest links when posting to chat.
